This repository is the ongoing RESTful Web API evolution of my previous project, [Worker-Availability-Checker](github.com/VladyslavShaposhnikov/Worker-Availability-Checker). I have transitioned the original console application into a web-based service to provide better accessibility, scalability, and performance.
    
Tech Stack & Key Features

    Framework: .NET 10 (C# 14) & ASP.NET Core Web API
  
    Database: SQL Server via Entity Framework Core (using Configuration classes for a clean architecture)
  
    Security: JWT Bearer Authentication & ASP.NET Core Identity
  
    Access Control: Role-based management
  
    Documentation: OpenAPI with Scalar for interactive API testing
  
    Design Patterns: Dependency Injection and modular configuration
